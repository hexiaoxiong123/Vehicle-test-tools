# 文本分析工具

## 功能说明

`text_frequency_analyzer.py` - 文本字符频率分析工具

## 功能

统计文本文件中每个字符出现的频率，按出现次数降序排列。

## 应用场景

### 在汽车测试中的应用

1. **日志关键字分析**
   - 统计日志中ERROR、WARN等关键字出现频率
   - 识别高频问题模式

2. **CAN报文分析**
   - 分析报文中某些字段的分布
   - 识别异常数据模式

3. **测试数据分析**
   - 统计测试结果中的字符分布
   - 数据质量检查

## 使用方法

```bash
# 基本用法
python text_frequency_analyzer.py -i test.txt

# 指定编码
python text_frequency_analyzer.py -i test.txt -e GBK
```

## 技术要点

- 使用字典统计字符频率
- Lambda表达式排序
- 文件编码处理
- 异常处理

## 扩展功能

可以扩展为：
- 词频统计（不是字符）
- Top N 高频词分析
- 导出为CSV/JSON格式
- 可视化展示（词云图）
